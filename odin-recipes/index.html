Odin Recipes

Odin recipes is a page built as a practice for learning web development in the odin project (TOP). The Odin Project is an open-source project that helps people learn about programming most notably Web development in both front-end and back-end. It is a useful go-to site for those enthusiasts who also want to learn other best tools and practices in programming. You can visit The Odin Project and get more info.

Odin Recipes comes as an individually built page for practicing the basics of HTML and CSS markup languages. Recipes mentioned here are drawn from the inspiration of my locality Tanzania. These recipes are delicious foods and drinks originally made in this sweet and cool East African spot. Check out these recipes and, if you personally feel like you want to taste them where they are made let me say "Karibu sana" meaning "Welcome".

The recipes are:

  • Ugali (cassava flour stiff porridge)
  • Ndizi Nyama (stewed bananas with meat)
  • Juisi ya ubuyu (baobab fruit juice)
